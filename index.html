Writing Idiomatic Python

By Mohammad Faizal under Python on 24 August 2013

When i started coding python,mostly i used for loop for complex iteration.Then i found awesome python built-in which solved most of iteration issues and gives pretty speed. Tips for Iteration in Python:

Map

Use Map function for iterating iterables to get manipulation for same set of them.
Eg: To get squares of each item list [1,2,3]

Reduce

Use reduce for iterating iterables to perform manipulation but finally we get single value from iterables.
Eg: To get the product of all items in a list [2,3,4].

Filter

The function name itself has meaning.Iterate the iterables to filter that.
Eg: To filter value which is lesser than 5 from a list [1,2,3,4,5,6,7,8,9]
